# 决策树

我们团队的小红还没男朋友，她妈妈正在给她张罗相亲的事，那我们来看看他们的对话：

- 小红：多大年纪了？
- 妈妈：26。
- 小红：长得帅不帅？
- 妈妈：挺帅的。
- 小红：收入高不？
- 妈妈：不算高，中等情况。
- 小红：是公务员不？
- 妈妈：是，在税务局上班呢。
- 小红：那好，我去见见。
